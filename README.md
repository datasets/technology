# Technology Datasets

Curated datasets on technology adoption, costs, and infrastructure — published on [DataHub](https://datahub.io/technology).

## Datasets

| Dataset | Description |
|---------|-------------|
| [gcat-artificial-space-objects](gcat-artificial-space-objects/) | Every artificial object ever launched into space — 68,000+ satellites, rocket bodies, and debris from 1957 to present |
| [genome-sequencing-costs](genome-sequencing-costs/) | Cost per genome from 2001 to present — a decline faster than Moore's Law |
| [historical-adoption-of-technology](historical-adoption-of-technology/) | Adoption curves for 100+ technologies across 150+ countries since 1800 |
| [cloud-providers-pricing](cloud-providers-pricing/) | Historical object storage pricing from AWS, GCP, and Azure |
| [adoption-rates-technology](adoption-rates-technology/) | US household adoption rates for key technologies from radio to smartphones |
